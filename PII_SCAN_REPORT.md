# LAB MATCH AI 공개 파일 개인정보 스캔 보고서

## 검사 범위

최종 ZIP에 포함되는 HTML, JavaScript, Markdown, Python과 숨김 파일을 검사했다. 원본 백업 파일과 중첩 ZIP은 최종본에 포함하지 않았다.

## 검사 결과

| 검사 항목 | 결과 |
|---|---:|
| 이메일 주소 형식 | 0건 |
| 메일 링크 스킴 링크 | 0건 |
| 명확한 국내 전화번호 형식 | 0건 |
| 전화 링크 스킴 링크 | 0건 |
| 연락처 전용 데이터 필드 | 0건 |
| 교수 사진 및 프로필 이미지 파일 | 0개 |
| 최종본 내부 중첩 ZIP | 0개 |
| 깨진 상대 경로 | 0건 |

`이메일`, `전화번호`와 같은 안내 문구 자체는 데이터 정책 문서에 남아 있지만 실제 연락처 값은 포함하지 않는다.

## 제거 대상 검사 기준

- 이메일 주소 정규식
- 국내 지역번호와 휴대전화 형식
- 메일 링크 스킴와 전화 링크 스킴 링크
- `email`, `emails`, `official_emails`
- `phone`, `telephone`, `mobile`, `fax`
- 사진 및 프로필 이미지 전용 필드
- 백업 및 중첩 압축파일

## 유지한 예외

다음은 서비스 목적에 필요해 유지했다.

- 대학과 학과의 공식 URL
- 교수 공식 프로필 URL
- 연구실 공식 홈페이지 URL
- 링커리어와 기업 공식 채용공고 URL
- GitHub Issues 정정 및 삭제 요청 URL

공식 URL은 수정 전후 목록과 순서가 동일한지 별도로 비교했다.
