# DGIST 학과 우선 탐색 개편

## 변경 구조

DGIST 연구실 탐색을 넓은 분야 배너에서 다음 2단계 구조로 변경했다.

1. 화학물리학과
2. 전기전자컴퓨터공학과
3. 뇌과학과
4. 로봇및기계전자공학과
5. 에너지공학과
6. 뉴바이올로지학과(New Biology)

학과를 먼저 선택하면 해당 학과의 세부 분야 배너와 검색창이 활성화된다. 사용자는 세부 분야 배너를 누르거나 선택 학과 안에서 연구 키워드를 직접 입력할 수 있다.

## 추천 알고리즘 변경

- 배너 추천 결과는 선택한 학과 교수님으로 제한한다.
- 직접 입력 검색도 선택 학과 내부에서만 추천한다.
- 직접 관련 결과, 인접 분야 결과와 교수명 직접 검색도 선택 학과 범위를 유지한다.
- 기존 검색에서 정상적으로 결과가 나오던 점수 계산과 카드 구조는 유지한다.
- 초기화하면 학과 선택 단계로 돌아간다.

## New Biology

뉴바이올로지학과를 독립 학과로 추가하고 다음과 같은 세부 분야를 제공한다.

- 유전체·DNA 복구
- 암·정밀의학
- 면역·감염
- 노화·대사
- 줄기세포·재생의학
- 단백질 구조·공학
- 화학생물학·신약
- 단백질체·오믹스
- 단분자·세포 이미징
- 나노바이오·전자약
- 식물생명·환경응답
- 동물생태·행동

## 통계 연속성

기존 페이지뷰, 검색 실행, 검색 결과 수, 더보기, 공식 링크, 만족도와 검색 복구 이벤트는 그대로 유지된다. 따라서 개편 전후 DGIST 전체 검색 결과 0개 비율과 공식 링크 클릭 지표를 같은 방식으로 비교할 수 있다.

새 구조의 성능을 학과와 세부 분야 단위로 분석하기 위해 다음 익명 집계만 추가했다.

- 학과 선택 횟수
- 학과별 세부 분야 선택 횟수
- 학과별 검색 결과 수 구간
- 세부 분야별 검색 결과 수 구간
- 학과별 공식 링크 클릭 횟수
- 세부 분야별 공식 링크 클릭 횟수

검색어 원문, 교수명, 사용자 ID와 개인별 이동 경로는 수집하지 않는다.

분석기는 다음 CSV를 추가 생성한다.

- `dgist_department_choice_summary.csv`
- `dgist_subfield_choice_summary.csv`
- `dgist_department_performance.csv`
- `dgist_subfield_performance.csv`

과거 데이터에는 새 이벤트가 없으므로 새 CSV가 비어 있는 것이 정상이다. 기존 지표는 과거 데이터도 그대로 분석된다. 학과·세부 분야별 성과는 이 수정본을 배포한 시각 이후부터 측정된다.

## 검증 결과

- 여섯 학과가 모두 표시되는지 확인
- 학과를 선택하기 전 검색창이 비활성화되는지 확인
- 학과별 대표 세부 분야에서 해당 학과 교수님만 표시되는지 확인
- 뉴바이올로지학과의 12개 세부 분야 표시 확인
- 초기화 후 학과 선택 화면으로 복귀하는지 확인
- JavaScript와 Python 분석기 구문 검사 통과
- 기존 GoatCounter export 분석 성공
- 신규 학과·세부 분야 outcome 및 공식 링크 이벤트의 합성 데이터 파싱 성공
