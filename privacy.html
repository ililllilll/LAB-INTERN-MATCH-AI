이용 통계 안내

LAB MATCH AI는 서비스 개선을 위해 사용자가 선택한 경우에만 집계형 이용 통계를 전송합니다. 동의를 거부해도 인턴 및 연구실 추천 기능을 모두 이용할 수 있습니다.

처리 목적

서비스별 이용 비중, 검색 결과 부족 또는 과다 구간, 공식 링크 이동, 기능 사용 빈도와 익명 만족도 응답을 분석하여 검색 기능과 화면 구성을 개선합니다.

집계하는 항목

  • 방문한 페이지 경로와 페이지 조회 횟수
  • 서비스 및 학교 선택, 버튼 클릭, 검색 실행, 필터, 더보기, 공식 링크 이동 횟수
  • 검색어 원문을 브라우저에서 넓은 연구 분야 범주로 변환한 값
  • 검색 결과 수 구간, 클릭 순위 구간, 도움이 됐어요 또는 아쉬워요 응답

저장하지 않는 항목

  • 검색어 원문
  • 이름, 학번, 이메일, 전화번호
  • 로그인 계정 또는 운영자가 생성한 사용자별 고유 ID
  • 개인별 전체 이동 경로

통계 도구와 접속 정보 처리

통계 도구로 GoatCounter를 사용합니다. GoatCounter는 추적용 쿠키나 사용자 식별자를 브라우저에 저장하지 않습니다. 중복 방문 판단 과정에서 사이트명, IP 주소, User-Agent 조합을 최대 8시간 동안 메모리에서 일시적으로 처리하지만 IP 주소와 전체 User-Agent는 데이터베이스에 저장하지 않습니다. 집계 데이터는 핀란드와 독일 소재 서버에서 처리되며 GoatCounter 운영자는 아일랜드에 있습니다.

선택 변경

아래에서 언제든 이용 통계 동의를 허용하거나 거부할 수 있습니다. 선택값은 현재 브라우저에만 저장되며 분석용 사용자 ID로 사용되지 않습니다.

운영 원칙

GoatCounter의 개별 페이지뷰 저장 기능은 사용하지 않고 집계 통계만 이용합니다. 수집 목적에 필요하지 않은 상세 정보는 수집하지 않으며, 분석 결과는 서비스 개선과 프로젝트 성과 검증에만 사용합니다.
