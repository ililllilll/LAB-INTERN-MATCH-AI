# DGIST 로봇및기계전자공학과 공식자료 기반 추천 재구축 보고서

## 1. 작업 범위

- 기준 파일: 사용자가 제공한 `LAB_MATCH_AI_Field_Search_Precision_Fix_Final(1).zip`
- 수정 대상: **학교별 연구실 추천 → DGIST → 로봇및기계전자공학과**
- 비대상: DGIST의 나머지 5개 학과, 분야별 연구실 추천, KAIST·서울대·POSTECH, 인턴 추천
- 원본 `graduate/dgist/data.js`: 수정하지 않음
- `analytics.js`: 수정하지 않음

## 2. 문제 원인과 변경 원칙

기존 로봇기계학과 추천은 공식 연구 분야와 함께 자동 수집 문장, 논문 제목, 넓은 통합 태그가 같은 점수층에 섞일 수 있었습니다. 이번 버전에서는 원본 데이터 파일을 보존하면서 로봇기계학과에만 별도의 정제 오버레이와 검색기를 적용했습니다.

추천 점수에는 다음 자료만 사용합니다.

1. DGIST 공식 교수 목록의 연구 분야
2. DGIST 대학원 연구실 안내의 공식 연구실명·연구 분야
3. 위 공식 내용을 바탕으로 수동 정리한 직접 검색어·관련어·분야 intent
4. 배너별 검증된 교수 목록

기존 자동 크롤링 키워드, 논문 제목, 넓은 `domainTags`와 다른 학과용 통합 태그는 로봇기계학과 전용 검색 점수에서 제외했습니다.

## 3. 구축 내용

- 공식 교수 28명 전원에 대해 별도 정제 프로필 작성
- 공식 연구실명 표시 오버레이 적용
- 교수별 핵심 연구 분야, 직접 검색어, 관련 검색어, 세부 intent 정리
- 로봇기계학과 선택 시에만 전용 검색기 활성화
- 직접 검색 결과 최대 10명 제한
- 여러 구체 개념이 함께 들어오면 모두 만족하는 교수만 추천
- `UAV`, `BMI`, `SLAM` 등 짧은 영문 표현은 단어 경계 기준으로 판정
- `웨어러블 로봇 센서`처럼 결합된 자연어는 `웨어러블 로봇 AND 웨어러블 센서`로 해석
- 세부 분야 버튼은 키워드 재검색이 아니라 공식 분야 기반 교수 화이트리스트로 연결
- 동일 분야 내에서는 공식 전문성이 더 직접적인 연구실이 먼저 오도록 제한적 우선순위 적용
