# LAB MATCH AI 통계 기능 보완 요약

## 기존 기능으로 가능했던 분석

- 서비스 및 학교 선택 비중
- 학교별 검색 실행량
- 결과 0개 및 결과 11개 이상 비율
- 더보기, 인접 분야, 공식 링크 및 공고 링크 클릭량
- 결과 순위 구간별 클릭량

## 이번에 추가한 기능

- 검색어 원문 대신 브라우저 안에서 익명 분야 범주만 분류
- 학교 및 분야별 검색 실패율과 결과 과다 비율 분석
- 검색 결과 아래 `도움이 됐어요`, `아쉬워요` 익명 피드백 추가
- 피드백 질문 표시 횟수도 기록해 응답률 계산 가능
- GoatCounter JSON export를 분석하는 Python 스크립트 추가
- 수정일 기준 전후 비교와 두 비율 검정 지원

## 개인정보 보호

- 검색어 원문, 이름, 학번, 이메일, 전화번호를 전송하지 않음
- 사용자 ID나 자체 추적 ID를 생성하지 않음
- 교수명과 기업명을 이벤트 이름에 기록하지 않음
- 기존 GoatCounter 대시보드와 주소만 사용

## 기존 서비스 유지

- 추천 데이터와 추천 알고리즘은 변경하지 않음
- 기존 pageview 집계와 누적 기록 유지
- `data.js`, `app.js` 분리 구조 유지
