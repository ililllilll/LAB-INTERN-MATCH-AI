## 4. 검색 알고리즘

### 4.1 배너 검색

18개 세부 분야 배너는 검증된 교수 목록에 직접 연결했다. 배너를 긴 문장으로 다시 검색하지 않으므로 같은 분야 배너가 학과 전체 교수로 확장되는 문제를 차단했다.

### 4.2 직접 검색

- 정확한 직접 검색어와 공식 핵심 분야를 가장 강하게 반영
- `AI`, `HCI`, `UX`, `IoT` 같은 짧은 영문 토큰은 단어 경계가 있을 때만 일치
- 두 개 이상의 인식 가능한 개념을 입력하면 모든 개념을 만족하는 후보를 우선
- 넓은 검색어도 사용자 화면에는 최대 10명만 표시
- 배너와 직접 검색이 같은 교수별 정제 DB를 사용

## 9. 통계 해석

GoatCounter 계정, 동의 구조, 기존 이벤트 이름은 유지된다. 따라서 다음 DGIST 전전컴 전체 지표는 수정 전후 비교할 수 있다.

- 검색 직후 결과 0개 비율
- 검색 결과 수 구간
- 검색 100회당 공식 홈페이지·프로필 클릭
- 만족도와 심화 탐색 행동

전전컴 배너 구성이 18개로 재편되었으므로 개별 배너 ID별 장기 비교는 새 배포 시각 이후를 새 기준으로 삼는 것이 안전하다. 검색어 원문, 교수명, 사용자 ID, 개인별 이동 경로는 새로 수집하지 않는다.
