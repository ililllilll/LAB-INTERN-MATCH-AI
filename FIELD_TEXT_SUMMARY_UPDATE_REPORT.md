# 분야별 통합 탐색 문구 및 연구 소개 문장 수정

## 변경 사항

- 상단 DB 문구를 `4개 대학 통합 DB: 교수님 총 1,228명`으로 변경
- 관심 연구 키워드 예시에서 `반도체 공정 중 식각과 증착` 제거
- 검색창 예시에서 `전고체전지 고체전해질` 제거
- 검색 안내를 `연구 주제를 구체적으로 입력할수록 결과가 정교해집니다.`로 축약
- 검색 결과 전체 인원과 학교별 인원 단위를 `개`에서 `명`으로 변경
- 안지환 교수와 정재웅 교수 연구 소개 문장을 완결형으로 수정
- 통합 교수 카드의 연구 소개가 쉼표, 말줄임표, 명사구 또는 `연구하며`로 끝나지 않도록 표시 단계에서 문장 종결을 보정

## 검증

- 분야별 통합 DB 1,228명 유지
- JavaScript 문법 검사 통과
- 안지환 교수 및 정재웅 교수의 원본 학교 DB와 통합 DB 문구 동기화
- 통합 카드 1,228개의 표시용 연구 소개가 모두 `합니다.`, `입니다.` 등의 완결형 문장으로 끝나는지 검사
